# 🍬 Nassau Candy Factory Reallocation & Shipping Optimization Recommendation System

## 📌 Project Overview

This project analyzes the Nassau Candy Distributor dataset to improve shipping efficiency by predicting shipping lead time and recommending the most suitable factory based on historical sales and operational data.

The system uses Machine Learning and Data Analytics to help optimize factory allocation and reduce shipping delays.

---

## 🎯 Objectives

* Analyze distributor sales data.
* Predict shipping lead time using Machine Learning.
* Recommend the best factory for product allocation.
* Visualize business insights through an interactive dashboard.
* Improve operational efficiency using data-driven decision making.

---

## 📂 Dataset

Dataset Used:

* Nassau Candy Distributor Dataset

Important Columns:

* Product Name
* Region
* Ship Mode
* Sales
* Units
* Cost
* Gross Profit
* Factory
* Lead Time
* Profit Margin (%)

## 🤖 Machine Learning Models

The following regression models were trained and evaluated:

* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor

The Random Forest model was selected for prediction due to its better performance.
